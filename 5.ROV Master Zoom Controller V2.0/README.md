# 1.新增加的特性

- 使用`MCU` STM32F103C8T6 
- 使用[RT-Thread](https://www.rt-thread.org/)操作系统
- 使用RT9193-33 LDO为MCU提供3.3V电源
- 增加`CP2102`  USB转UART模块进行调试
- 使用ULN2803达林顿管驱动芯片（8通道）驱动步进电机
